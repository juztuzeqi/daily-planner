# Daily Routine Planner 📋

Aplikasi PWA untuk merencanakan rutinitas harian.

## ✨ Fitur
- ✅ Routine harian (muncul otomatis setiap hari)
- ✅ Appointment/Non-routine (khusus hari tertentu)
- ✅ Checklist dengan status done/undone
- ✅ Navigasi antar tanggal
- ✅ Bekerja offline
- ✅ Bisa diinstall seperti aplikasi native

## 🚀 Cara Install
1. Buka di Chrome Android
2. Klik menu ⋮
3. Pilih "Install App" atau "Add to Home screen"

## 🛠️ Teknologi
- HTML, CSS, JavaScript
- LocalStorage untuk penyimpanan data
- Service Worker untuk offline capability
- Manifest.json untuk PWA

## 📁 Struktur File
- `index.html` - Aplikasi utama
- `manifest.json` - Konfigurasi PWA
- `sw.js` - Service Worker
- `icon-192.png` - Icon 192x192
- `icon-512.png` - Icon 512x512
